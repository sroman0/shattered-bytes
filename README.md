# Shattered Bytes

Serious game per il corso di Computer Forensics and Cyber Crime Analysis.

## Comandi

```bash
npm run dev
npm run build
npm run preview
npm run levels:build
```

## Struttura

- `src/`: applicazione React.
- `public/`: asset serviti direttamente dal browser, inclusi livelli JSON e intro.
- `assets/`: asset sorgente usati per generare i livelli.
- `scripts/level_generation/`: generatori deterministici dei livelli.
- `scripts/experiments/`: script sperimentali o una tantum non usati dalla build.
- `docs/`: walkthrough, direttive e note di progetto.

Vedi `docs/PROJECT_STRUCTURE.md` per una mappa piu dettagliata.
